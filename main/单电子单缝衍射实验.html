单电子单缝衍射实验

实验背景

实验简介

单电子单缝衍射实验是验证微观粒子波粒二象性的关键实验。当电子流强度较大时，大量电子短时间内通过单缝，接收屏上直接呈现连续的衍射图样；当电子流强度调至很弱时，电子逐个通过单缝，屏上初始呈现随机分布的亮点（体现粒子性），但随着电子数量累积，逐渐形成与强电子流相同的衍射图样（体现波动性）。这一结果表明，波动性是每个电子的固有属性，而非电子间相互作用的结果。

实验描述

单电子单缝衍射实验是指将电子束强度调控到使电子逐个通过单缝，在接收屏上逐步积累形成衍射图样的物理实验。该实验直接展示了微观粒子的波粒二象性：单个电子在屏上呈现随机点（粒子性），大量电子的统计分布形成确定的衍射图样（波动性）。

教学提示

本实验的核心是波函数的统计解释：波函数在空间中某一点的强度（振幅绝对值的平方）决定在该点找到粒子的概率密度，衍射图样即为概率分布的宏观体现。实验表明，电子的波动性既表现为大量电子在同一实验中的统计结果，也表现为单个电子在许多次相同实验中的统计结果。单个电子落点随机，但大量电子的统计分布形成确定的衍射图样——这是理解单电子衍射现象的关键。

实验原理

实验原理

1. 微观粒子的波粒二象性

1924年，德布罗意提出微观粒子具有波粒二象性的假说，将粒子的性质与波的性质通过以下关系联系起来（正像光子和光波的关系一样）：

\(E = h\nu = \hbar\;\omega \quad\quad \pmb{p} = \dfrac{h}{\lambda}\pmb{e}_{\pmb{\text{k}}} = \hbar\;\pmb{k}\)

其中 \(h\) 为普朗克常量，\(\hbar = \frac{h}{2\pi}\)为约化普朗克常量，\(\omega\)为波的角频率，\(\pmb{k}\)为波矢（大小\(k = \frac{2\pi}{\lambda}\)），\(\pmb{e}_{\pmb{\text{k}}}\)为波矢方向的单位矢量。该公式称为德布罗意公式，或德布罗意关系。

2. 波函数的统计解释

  • 波函数的概率解释由玻恩首先提出。他指出，波函数在空间中某一点的强度（振幅绝对值的平方）与该点找到粒子的概率密度成正比。按照这种解释，描述微观粒子的波乃是概率波。
  • 在衍射图样中，衍射极大处波的强度大，粒子出现在该处的概率也大，因而投射到该处的粒子多；衍射极小处波的强度很小（甚至为零），粒子出现在该处的概率也很小（或为零），因而投射到该处的粒子很少（或没有）。因此，由波函数可得出粒子在空间任意一点出现的概率，波函数完全描述了微观粒子的量子态。

3. 单电子衍射的积累现象

实验表明，衍射图样的分布与入射电子流强度无关。当电子流强度减小到使电子逐个通过单缝（同一时刻仅有一个电子）时，接收屏上最初出现随机分布的点，体现出粒子性。但随着时间增加，点的数目逐渐增多，其分布逐渐形成与强电子流时完全相同的衍射图样。这说明单个电子的行为与其他电子无关，衍射图样并非由电子间的相互作用产生。实验所体现的电子的波动性，是大量电子在同一实验中的统计结果，也是单个电子在许多次相同实验中的统计结果。单个电子的落点看似随机，但其概率分布是确定的。

4. 概率波的数学描述

  • 设波函数 \(Φ(\pmb{r},t)\) 描写粒子的状态。以 \(\mathrm{d}W(\pmb{r},t)\) 表示在时刻 \(t\) ，体积元 \(\mathrm{d}V = \mathrm{d}x\mathrm{d}y\mathrm{d}z\)内找到粒子的概率。按照波函数的统计解释，有
    \[\mathrm{d}W(\pmb{r},t) = C|Φ(\pmb{r},t)|^{2}\mathrm{d}V\]
    式中\(C\)为归一化常数。概率密度定义为
    \[w\left( \pmb{r},t \right) = \frac{\mathrm{d}W\left( \pmb{r},t \right)}{\mathrm{d}V} = C|Φ(\pmb{r},t)|^{2}\]
    它表示在时刻\({t}\)、\(\pmb{r}\)点附近单位体积内找到粒子的概率。
  • 由于粒子必定要在空间某点出现，粒子在空间各点出现的概率总和等于1，因此波函数满足归一化条件：
    \[C\int_{- \infty}^{+ \infty}{\left| Φ(\pmb{r},t) \right|^{2}\mathrm{d}V} = 1\]
    或等价地，归一化波函数满足\(\int_{- \infty}^{+ \infty}{|Ψ|^{2}\mathrm{d}V} = 1\)。

5. 实验现象

  • 改变入射电子总数：在电子流强度（单位时间电子数）固定时，入射电子总数较少时，接收屏上仅出现少量随机分布的点，无法形成完整图样；随着入射电子总数增加，点的数目逐渐增多，逐步呈现衍射条纹的轮廓；当入射电子总数足够多时，形成清晰稳定的衍射图样。
  • 改变电子流强度：在入射电子总数固定时，当电子流强度较大时，单位时间内通过单缝的电子多，接收屏上很快出现完整的衍射图样；当电子流强度较小时，电子逐个通过，接收屏上先出现一个个随机分布的点，需要更长的实验时间才能积累形成与强电子流时完全相同的衍射图样。

注：衍射图样的稳定性和清晰度仅取决于入射电子总数是否达到足够的统计数量，与电子流强度无关。电子流强度只影响图样形成的快慢，不改变衍射条纹的位置和相对强度。

原理应用

基于电子衍射图样与缝宽、德布罗意波长的关系，该实验的原理可应用于纳米尺度的精密尺寸测量、量子力学基本原理论证、电子光学系统分析等领域。

学习要点： 波函数的统计解释是理解本实验的核心：波函数在空间某点的强度（模平方）决定在该点找到电子的概率密度。单个电子落点随机，但大量电子的统计分布形成确定的衍射图样，这是理解单电子衍射现象及其应用的理论基础。

操作方法

操作方法

1

调节入射电子总数：滑动"入射电子总数"滑块或在数值输入框中输入电子数量。

观察要点：保持电子流强度（单位时间电子数）不变，当入射电子总数较少时，接收屏上出现一个个随机分布的点，体现电子的粒子性；随着电子总数逐渐增加，点的分布逐渐呈现衍射条纹的轮廓；当电子总数足够多时，形成清晰的衍射图样，体现电子的波动性。这一过程表明，衍射图样是大量电子在同一实验中的统计结果。
2

调节电子流强度：滑动"入射电子流强度"滑块或在数值输入框输入单位时间电子发射数。

观察要点：保持入射电子总数不变，当电子流强度较大时，单位时间内打到接收屏上的电子多，衍射图样迅速形成；当电子流强度较小时，电子逐个通过单缝并到达接收屏，需要更长时间积累才能形成衍射图样，但最终形成的图样与电子流强度较大时完全相同。这一结果表明，衍射图样的分布与电子流强度无关，仅取决于入射电子总数是否达到足够的统计数量。

操作提示： 建议按步骤依次操作，采用控制变量法：步骤①固定电子流强度、改变入射电子总数；步骤②固定入射电子总数、改变电子流强度。通过对比观察，理解"入射电子总数决定图样完整性，电子流强度只影响图样形成速度"的规律。
